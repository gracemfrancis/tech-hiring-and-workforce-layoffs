# tech-hiring-and-workforce-layoffs
First year undergrad project. An interactive Python tool analyzing  25 years of tech industry layoffs and hiring trends (2000–2025).  Features data visualization, Random Forest predictions, KMeans  clustering, and Gradient Boosting forecasts. Built with pandas,  NumPy, matplotlib, seaborn, and scikit-learn.
A dynamic, menu-driven Python application that analyzes 25 years of 
tech workforce data — covering hiring trends, layoffs, and job market 
shifts across industries and countries. Built with pandas, NumPy, 
matplotlib, seaborn, and scikit-learn, the tool offers five modes of 
analysis: a global overview, single-year deep dives, year range 
comparisons, ML-powered predictions (Random Forest, Gradient Boosting, 
KMeans clustering), and a job market intelligence dashboard. All 
analyses generate publication-quality charts saved automatically as 
PNG files. Designed as an everyday reference tool for students, 
researchers, and anyone tracking the state of the tech job market.
